Как создать сайт бесплатно за один день: GitHub + Vercel + Claude

Инструменты AI-маркетинг Vibe coding Сайт бесплатно
Я основатель школы маркетинга Target School, 17 лет в интернет-маркетинге. Всё это время я зависела от разработчиков для каждого технического шага.

В апреле 2026 года я создала полноценный сайт с нуля за один день. С главной страницей, лендингом продукта, блогом, SEO и аналитикой. Стоимость — 0 рублей.

Вот точная инструкция как это повторить.
🛠
Три инструмента. Все бесплатные.
01
Claude
claude.ai
Пишет весь код по вашим словам. Не нужно знать программирование — просто объясняете что хотите.
02
GitHub
github.com
Хранит файлы вашего сайта. Представьте это как папку в интернете, которая всегда доступна.
03
Vercel
vercel.com
Берёт файлы из GitHub и публикует их как живой сайт. Автоматически, за 30 секунд.
01

02
Создайте репозиторий на GitHub

Репозиторий — это папка для файлов вашего сайта на GitHub.

GitHub — создание репозитория
github.com/new

Create repository
  1. Зайдите на github.com под своим аккаунтом
  2. Нажмите зелёную кнопку New в левом углу или знак + вверху справа
  3. В поле Repository name напишите название: например my-site
  4. Выберите Public — это обязательно для бесплатного хостинга
  5. Нажмите Create repository
03
Создайте страницу через Claude

Это главный шаг. Вы просто разговариваете с нейросетью как с человеком.

Откройте claude.ai и напишите примерно так:

Пример запроса к Claude
Создай главную страницу сайта для [ваша профессия]. Я [ваше имя], [что вы делаете]. Стиль минималистичный, современный. Нужны: — шапка с навигацией — героблок с заголовком и кнопкой — раздел обо мне с цифрами — футер со ссылками на соцсети Сделай готовый HTML-файл.
Как уточнять результат: «Поменяй шрифт на более элегантный» / «Сделай фон потеплее» / «На мобильном выглядит плохо — исправь» / «Добавь блок с услугами». Говорите как с человеком — технические термины не нужны.
Когда результат устроит — скачайте файл. Переименуйте его в index.html — именно это имя GitHub распознаёт как главную страницу.
04
Загрузите файл на GitHub

Commit changes
  1. Зайдите в ваш репозиторий на GitHub
  2. Нажмите Add file → Upload files
  3. Перетащите файл index.html в окно
  4. Нажмите зелёную кнопку Commit changes
05
Опубликуйте сайт через Vercel

Это самый быстрый шаг. 30 секунд — и сайт живой.

Vercel — деплой проекта
vercel.com/new
Выберите репозиторий
my-site
Import
Готово к публикации
Настройки менять не нужно
🚀 Deploy
my-site.vercel.app ✓
  1. Зайдите на vercel.com
  2. Нажмите Add New → Project
  3. Найдите в списке ваш репозиторий — нажмите Import
  4. Ничего не меняйте в настройках
  5. Нажмите Deploy
Магия системы: теперь каждый раз когда вы загружаете новый файл на GitHub — Vercel автоматически обновляет сайт. Без дополнительных действий.
06
Подключите свой домен

Если у вас есть свой домен — подключить его просто. Если нет — сайт уже работает на адресе Vercel.

  1. В Vercel: Settings → Domains → введите ваш домен → Add
  2. Vercel покажет DNS-записи которые нужно добавить
  3. Зайдите к регистратору домена (Cloudflare, Namecheap, reg.ru)
  4. Добавьте указанные записи в раздел DNS
  5. Подождите 5-10 минут — домен подключён
07
Добавьте новые страницы блога

Каждая новая страница — это просто новая папка в репозитории.

Запрос к Claude для новой страницы
Создай страницу статьи в том же стиле что и главная. Тема: [название статьи] Теги: [теги] Время чтения: [минуты] Контент: [текст статьи]
  1. Попросите Claude создать страницу — получите HTML-файл
  2. На GitHub нажмите Add file → Create new file
  3. В поле имени напишите: название-папки/index.html
  4. Как только напишете / — GitHub сам создаст папку
  5. Вставьте код от Claude → Commit changes
Страница будет доступна по адресу: ваш-сайт.com/название-папки
08
Настройте SEO и аналитику

Чтобы сайт находили в Google и Яндексе — попросите Claude добавить нужные элементы одним запросом.

Запрос к Claude
Добавь в мой index.html: — мета-теги title и description для SEO — Open Graph теги для красивых карточек в соцсетях — Яндекс.Метрику с кодом: [код счётчика] Вот мой файл: [вставьте код]
  1. Зарегистрируйтесь на metrika.yandex.ru — создайте счётчик
  2. Попросите Claude добавить код метрики во все страницы
  3. Попросите Claude создать sitemap.xml и robots.txt
  4. Загрузите все файлы в корень репозитория на GitHub
  5. Зарегистрируйте сайт в Google Search Console и Яндекс.Вебмастере
